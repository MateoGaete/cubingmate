# 🔍 Cómo Ver los Errores en el Navegador - Guía Simple

Cuando algo no funciona, el navegador guarda información sobre qué salió mal. Te explico cómo verla.

## 🎯 Método 1: Presionar F12 (Más Fácil)

### Paso a Paso:

1. **Abre tu tienda** en el navegador (Chrome, Edge, Firefox, etc.)
2. **Presiona la tecla F12** en tu teclado
   - Está arriba, en la fila de números
   - Si no funciona, intenta **Fn + F12**
3. **Se abrirá una ventana** en la parte inferior o lateral de tu navegador
4. **Busca la pestaña que dice "Console"** o **"Consola"** (en español)
5. **Haz clic en esa pestaña**
6. **Intenta hacer el pago** de nuevo
7. **Busca mensajes en ROJO** - esos son los errores
8. **Copia el mensaje completo** que aparece en rojo

## 🎯 Método 2: Clic Derecho (Alternativa)

Si F12 no funciona:

1. **Haz clic derecho** en cualquier parte de la página de tu tienda
2. **Busca la opción** que dice:
   - "Inspeccionar" o "Inspeccionar elemento"
   - "Inspect" (en inglés)
   - "Ver código fuente" o "Developer Tools"
3. **Haz clic en esa opción**
4. **Se abrirá la misma ventana** que con F12
5. **Busca la pestaña "Console"** o **"Consola"**
6. **Sigue los pasos 6-8 del Método 1**

## 🎯 Método 3: Desde el Menú del Navegador

### En Chrome o Edge:

1. **Haz clic en los tres puntos** (⋮) en la esquina superior derecha
2. **Busca "Más herramientas"** o **"More tools"**
3. **Haz clic en "Herramientas para desarrolladores"** o **"Developer tools"**
4. **Busca la pestaña "Console"**

### En Firefox:

1. **Haz clic en las tres líneas** (☰) en la esquina superior derecha
2. **Busca "Más herramientas"** o **"More tools"**
3. **Haz clic en "Herramientas de desarrollo web"** o **"Web Developer Tools"**
4. **Busca la pestaña "Consola"** o **"Console"**

## 📸 Qué Buscar

Cuando abras la consola, verás algo así:

```
┌─────────────────────────────────────────┐
│ Console                                 │
├─────────────────────────────────────────┤
│                                         │
│ Error: Access Token inválido            │ ← Esto es lo que buscas (en ROJO)
│    at [archivo].js:10                 │
│                                         │
│ Error: No hay productos en el carrito   │ ← Otro error (en ROJO)
│                                         │
└─────────────────────────────────────────┘
```

## ✅ Qué Hacer con el Error

1. **Lee el mensaje completo** que aparece en rojo
2. **Copia el texto** (selecciónalo y Ctrl+C)
3. **Compártelo conmigo** para que pueda ayudarte

## 🔍 Ejemplos de Errores Comunes

### Error 1: "Access Token inválido"
```
Error: Token inválido. Verifica que hayas configurado correctamente tu token.
```
**Significa**: Tu token no está bien configurado

### Error 2: "No hay productos en el carrito"
```
Error: No hay productos en el carrito
```
**Significa**: El carrito está vacío

### Error 3: "Faltan datos del cliente"
```
Error: Faltan datos del cliente (nombre o email)
```
**Significa**: No completaste todos los campos del formulario

## 💡 Consejos

- **No te preocupes** si ves muchos mensajes - busca los que están en **ROJO**
- **Los mensajes en gris o negro** generalmente son informativos, no errores
- **Si no ves nada en rojo**, el error puede estar en otra pestaña (Network/Red)

## 🆘 Si No Puedes Abrir la Consola

1. **Intenta en otro navegador** (Chrome, Edge, Firefox)
2. **Verifica que tu teclado funcione** (presiona otras teclas)
3. **Intenta el Método 2** (clic derecho)
4. **Dime qué navegador usas** y te ayudo con pasos específicos

- [ ] Vi mensajes en ROJO
